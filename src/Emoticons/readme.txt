Exercício 4 - Emoticons

Nesse exercício fiz primeiro o que dava para ler os emoticons, porém se colocasse os dois emoticons na mesma frase, a saida aparecia apenas o último emoticon digitado.
Então utilizei uma função para ele contar quantos emoticons continha na frase, para utilizar no if se o número de um emoticon é maior que o outro e assim, sair certo a maior quantidade de emoticons que contem em uma frase.
Ao fazer o contador tive que pesquisar em como fazer ele, e até eu entender como funcionava o contador, levei um tempo.
Então demorei mais que o planejado para conseguir resolver este exercício.
